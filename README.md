# RetroCheatEdit
RetroArchやJohn Emurator(Android)のchtファイルの編集を行うエディタです。<br>
<br>
RetroArchやJohn Emuratorの相互変換等の用途に使います。<br>
<br>
ppssppのチートファイル(*.ini)の編集にも対応させようと思ってます。
